About CosmosAIGraph

CosmosAIGraph implements the OmniRAG pattern using a powerful combination of Azure services and open-source technologies:

  • Azure Cosmos DB for NoSQL - Reliably stores and searches source data and session state/feedback
  • Azure Cosmos DB Vector Index with DiskANN - Stores and searches source data embeddings
  • Apache Jena - Stores entities and relationships in a graph structure

The solution orchestrates the flow between these three components based on detected user intent and provides Natural Language to Query (NL2Query) capability for graph queries with full SPARQL 1.1 support.
